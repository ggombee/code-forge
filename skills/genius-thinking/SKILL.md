# Genius Thinking Skill

> 10가지 인지 공식 + 검증된 혁신 프레임워크로 체계적 아이디어 생성

---

## 사용 시점

- 혁신 아이디어 발상 (새 서비스/제품)
- 복잡한 문제 해결 (다각도 접근)
- 창의적 돌파 (기존 접근 한계)
- 전략 설계 (장기 비전)
- 블루오션 기회 (제품 차별화)

---

## 10가지 핵심 공식

| # | 공식 | 프레임워크 | 핵심 |
|---|------|-----------|------|
| 1 | Genius Insight (GI) | JTBD | 관찰 깊이 + 독창성 + 패턴 인식 |
| 2 | Multidimensional Analysis (MDA) | Six Thinking Hats | 5차원(시간/공간/추상/인과/계층) × 6관점 |
| 3 | Creative Connection (CC) | SCAMPER | 개념 교차점 + 배타적 OR 패턴 |
| 4 | Problem Redefinition (PR) | HMW + ERRC | "How might we..." + 가치 혁신 |
| 5 | Innovative Solution (IS) | TRIZ 40 원칙 | 분할, 추출, 비대칭, 병합 등 |
| 6 | Insight Amplification (IA) | Incubation | 정의 → 휴식 → 통찰 포착 |
| 7 | Thought Evolution (TE) | - | 학습 → 실험 → 메타인지 |
| 8 | Complexity Solving (CS) | Morphological Box | Zwicky 형태학적 분석 |
| 9 | Intuitive Leap (IL) | DMN-ECN Switching | 확산적↔수렴적 사고 전환 |
| 10 | Integrated Wisdom (IW) | - | 지식+이해+지혜+행동 통합 |

---

## 워크플로우

| Phase | 작업 | 필수 |
|-------|------|------|
| **0** | 입력 인자 검증 (없으면 요청) | ARGUMENT 필수 |
| **1** | 공식 2개+ 선택 + 프레임워크 검증 | Sequential thinking |
| **2** | 심층 분석 (1500자+, HMW 변환, SCAMPER/TRIZ) | 1500자 이상 |
| **3** | 12개+ 초안 아이디어 (tree-of-thoughts 병렬) | 12개 이상 |
| **4** | 10개+ 최종 아이디어 + ERRC 검증 (3000자+) | 3000자 이상 |
| **5** | `.hypercore/genius-ideas/`에 저장 | 파일 저장 필수 |

---

## 필수 산출물

- 공식 선택 근거
- 심층 분석 (1500자+ HMW 변환 포함)
- 10개+ 아이디어 (제목, 설명, 혁신 포인트, JTBD 연결, 실행 단계, 1-5 점수)
- 우선순위화 (ERRC 검증, 퀵윈 vs 빅벳)
- 다음 단계 (1-3개 즉시 실행 가능 액션)

---

## 금지 사항

| 금지 |
|------|
| 인자 없이 시작 |
| 프레임워크 검증 없이 공식 사용 |
| HMW 변환 없이 문제 정의 |
| SCAMPER/TRIZ 추론 없이 아이디어 나열 |
| 10개 미만 아이디어 또는 3000자 미만 |
| 출처 없는 프레임워크 설명 |
| 저장 없이 종료 |
