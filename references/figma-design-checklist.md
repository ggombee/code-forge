# Figma Design Checklist

스크린샷 상세 분석 체크리스트와 Figma vs 코드 비교 테이블 템플릿.

---

## 스크린샷 상세 분석 체크리스트 (필수)

### 1. 버튼/드롭다운 컴포넌트

- [ ] 기본 상태: 배경색, 테두리색, 텍스트색
- [ ] 선택 상태: 배경색, 테두리색, 텍스트색 (다른지 확인!)
- [ ] 아이콘 유무: 있으면 어떤 아이콘? 위치는? 색상은?
- [ ] 아이콘도 상태별로 색상 변하는지?

### 2. 액션 버튼 (초기화, 적용 등)

- [ ] 아이콘 + 텍스트 조합인지, 텍스트만인지?
- [ ] 아이콘 종류: ↻(리셋), ✕(닫기), +(추가) 등
- [ ] 위치: 좌측/우측/중앙

### 3. 레이아웃

- [ ] 가로 정렬 방향 (flex-direction)
- [ ] 간격 (gap) 값
- [ ] 줄바꿈 시 동작 (flex-wrap)
- [ ] 줄바꿈 시 아이템 정렬 (align-items)

## Figma vs 코드 비교 테이블 템플릿

### 코드에서 실제 스타일 값 추출

```bash
# 해당 컴포넌트의 styled 정의 확인
rg "styled\." {컴포넌트파일} -A 10

# 사용 중인 색상/스타일 상수 확인
rg "semanticColor|palette|#[0-9a-fA-F]" {컴포넌트파일}
```

### 컴포넌트별 상태 비교 테이블

| 컴포넌트 | 속성 | 상태 | Figma | 코드 | 일치 |
|----------|------|------|-------|------|:----:|
| 필터버튼 | 테두리색 | 기본 | ??? | ??? | ? |
| 필터버튼 | 테두리색 | 선택 | ??? | ??? | ? |
| 필터버튼 | 텍스트색 | 기본 | ??? | ??? | ? |
| 필터버튼 | 텍스트색 | 선택 | ??? | ??? | ? |
| 필터버튼 | 아이콘색 | 선택 | ??? | ??? | ? |
| 초기화버튼 | 아이콘 | - | 있음/없음 | 있음/없음 | ? |
| 초기화버튼 | 아이콘종류 | - | ↻/✕/기타 | ??? | ? |
| 레이아웃 | gap | - | ???px | ???px | ? |
| 레이아웃 | flex-wrap | - | wrap | ??? | ? |

**불일치 = 반드시 작업 내용에 포함**

### Figma vs 코드 비교

## Figma 일치 검증 체크리스트 (완료 시)

- [ ] 스크린샷 다운로드 완료
- [ ] 선택/미선택 상태 색상 일치
- [ ] 버튼 스타일 (색상, 아이콘) 일치
- [ ] 간격/패딩 값 일치
- [ ] hover/active/disabled 상태 일치

---

## Pixel-perfect 체크리스트 (figma-to-code)

- [ ] 레이아웃 (flex-direction, gap, padding) 일치
- [ ] 색상 (semantic color 매핑) 일치
- [ ] 폰트 (size, weight, line-height) 일치
- [ ] 상태별 스타일 (hover, active, disabled) 일치
- [ ] 반응형 동작 (있는 경우)
